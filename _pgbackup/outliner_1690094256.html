Outliner - a drawing outline tool

Outliner is a program I am working on to aid in art pedagogy.  Inspired by the writings of John Ruskin, which has the reader draw various plant outlines to train the hand in correct proportion, angle, and shape representation, this program helps by superimposing a grid over pattern images and the user's drawing space to break the exercises up into smaller, more manageable pieces.  It also calculates the artist's accuracy in copying the figure's outline in real time, pixel by pixel, an analog to Ruskin's recommendation for using tracing paper and comparing one's drawing to the original for accuracy.  

My intention with this program is to make the process of learning hand/eye coordination when drawing figure outlines a more seamless process.  I intend to refine this project by including a pre-defined catalog of pattern images and working on centralizing the drawing space rather than having the current split-screen iteration, perhaps implementing a zoom function for the pattern image.  This was done in C++ using SFML.
