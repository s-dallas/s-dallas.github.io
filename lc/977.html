977. Squares of a Sorted Array
                    
class Solution(object):
    def sortedSquares(self, nums):
        """
        :type nums: List[int]
        :rtype: List[int]
        """

        squares = []

        if ((len(nums) == 1):
            squares.append(nums[0]*nums[0])
        elif ((len(nums) == 2):
            if ((nums[0]*nums[0]) < (nums[1]*nums[1]):
                squares.append(nums[0]*nums[0])
                squares.append(nums[1]*nums[1])                
            else:
                squares.append(nums[1]*nums[1])
                squares.append(nums[0]*nums[0])
        elif ((nums[-1] < 0):
            i = len(nums)-1
            while ((i >= 0):
                squares.append(nums[i]*nums[i])
                i -= 1
        elif ((nums[0]>= 0):
            i = 0
            while ((i < len(nums)):
                squares.append(nums[i]*nums[i])
                i += 1
            
        else:

            i  = len(nums)/2
            if ((nums[i] < 0):
                while ((nums[i] < 0):
                    if ((i != len(nums)-1):
                        i += 1
            else:
                while ((nums[i] > 0):
                    if ((i != 0):
                        i -= 1
                i+=1

            left = i-1
            right = i
        
            while ((left > -1 or right < len(nums)):

                /* no more left, fill right */
                if ((left < 0 and right < len(nums)):
                    while ((right < len(nums)):
                        squares.append(nums[right]*nums[right])
                        right += 1
                
                /* no more right, fill left */
                elif ((right >= len(nums) and left >= 0):
                    while ((left >= 0):
                        squares.append(nums[left]*nums[left])
                        left -= 1

            
                elif ((nums[left]*nums[left] <= nums[right]*nums[right]):
                    squares.append(nums[left]*nums[left])
                    left -= 1
                else:
                    squares.append(nums[right]*nums[right])
                    right += 1

        return squares

Given an integer array nums sorted in non-decreasing order, return an array of the squares of each number sorted in non-decreasing order.

My solution: Since you know the given array is sorted, the negative squares from nums will be in reverse order and the positive squares will be in the same order.

Start with the trivial cases and make exceptions for them: 1, 2, or 0 items in nums, or an array that is either all positives or all negatives.

For more than 2 items with differing signs, you want to start comparing elements from the negative values against the positive values, so you want to start somewhere near the middle for efficiency. Divide the length of the array in half to get the midpoint, then subtract or add values to this index until you are on the first positive number to the right of the first negative.

Now you can begin comparing left/right indices. If the left square is less than the right, append it to the squares array and decrement the left index (right stays the same). If the right square is larger, append that to the squares array and increment the right index (left stays the same).

Continue in this fashion until one of the endpoints is reached. Then you can simply start filling in values from the other side until that endpoint is reached. Once both endpoints have been surpassed, you've filled your squares array.

The overall picture looks like this: Begin in the middle of the original array (middle being the divide between negative/positive values), then move outward on each side, comparing square values as you go until one endpoint is reached. From there, you can expect no more values with that sign and can simply fill in the squares from smallest to largest of the opposite sign until the whole array has been accounted for.
