1295. Find Numbers with Even Number of Digits

Given an array nums of integers, return how many of them contain an even number of digits.

My solution: This one combines modulus, division, and loops. Iterate through each number in the list/array, continually divide it by 10 and round down (math.floor). Each time, keep track of the digits. When you reach 0, break out of the loop and test if it's an even number of digits using mod 2. If so, increment the total count. In the end, simply return the final count.
