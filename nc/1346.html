1346. Check If N and Its Double Exist

Given an array arr of integers, check if there exist two indices i and j such that :

  • i != j

  • 0 <= i, j < arr.length

  • arr[i] == 2 * arr[j]

My solution: This one is not overly difficult. Nested for loops will get you to the solution. The outer loop iterates through the array, the inner loop does likewise, checking each subsequent element to see if it's double the value of the first pointer. Don't forget to take into account the stipulation that the indices must be different. There is one case where multiplying a value by 2 gives the same value, namely 0.
