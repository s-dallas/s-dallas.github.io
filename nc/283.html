283. Move Zeroes

Given an integer array nums, move all 0's to the end of it while maintaining the relative order of the non-zero elements.

Note that you must do this in-place without making a copy of the array.

My solution: This solution to this is very similar to problem 26.2. You want to use 2 pointers, only this time you will be advancing one through the array so long as it lands on a 0 rather than a duplicate. You can take into account the edge cases and do the swap here in much the same way you did there, and note that you don't need to return anything, only move the zeros to the end of the array.
